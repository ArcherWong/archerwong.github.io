我看不起的基础之人到中年

发表于 2017-06-08 | 分类于 随笔 | | 阅读次数

    从小到大，几乎所有的老师都会强调要注重基础。可是，年轻的我哪里会管什么基础，只觉得基础就是最简单的东西，会那么简单的东西有什么用，完全不能表现出我很强，所以我一向看不起基础。最近发现，好多事好多爱好，学了这么久，玩了这么久，一直都没什么进步。静下心来一想，我开始意识到基础确实挺重要的。
    今年，根据联合国的定义，92年的我居然步入了中年人的行列。刚开始看到的时候，觉得这就是个笑话一笑而过。现在想想，好像还真有一定的道理。当天赋开始老去，我确实不再是少年。曾经的空中楼阁虽然还在，但我开始明白我的天赋已经支撑不起更大的重量了。所以，中年人该拿出中年人的决心了，把当年扔掉的基础一个一个补回来，把破碎的空中楼阁推翻重建。
    PS：想了想，好像只有英语基础还可以（多谢王保长利忠先生）。既然现在在干前端，那就从前端开始吧。革命既然开始，同志应该努力了，加油！！！

各位老板，赏口吃的！
