我看不起的基础之人到中年

发表于 2017-06-08 | 分类于 随笔 | 阅读次数

    从小到大，几乎所有的老师都会强调要注重基础。可是，年轻的我哪里会管什么基础，只觉得基础就是最简单的东西，会那么简单的东西有什么用，完全不能表现出我很强，所以我一向看不起基础。最近发现，好多事好多爱好，学了这么久，玩了这么久，一直都没什么进步。静下心来一想，我开始意识到基础确实挺重要的。

前端模块化方案

发表于 2017-05-23 | 分类于 学习 | 阅读次数

一、模块化

    在没有模块化工具之前，我们想要引入多个js文件，需要在页面中插入一个一个的<script>标签。使用这种方式去管理 JavaScript 项目会有一些问题：

    1. 如果依赖不存在，或者引入顺序错误，应用程序将无法正常运行。

    2. 如果依赖被引入但是并没有使用，那样就会存在许多浏览器不得不下载的无用代码。

    而通过模块化工具，我们只需要声明模块所需的依赖，模块化工具就能够利用这些信息去构建依赖图表，然后使用图表生成一个优化过的，会以正确代码顺序被运行的 bundle。并且没有用到的依赖将不会被 bundle 引入。

二、模块化方案

    js语言本身并不支持模块化，同时浏览器中js和服务端nodejs中的js运行环境是不同的，如何实现浏览器中js模块化主流有两种方案：

    1. requirejs/seajs: 是一种在线“编译”模块的方案，相当于在页面上加载一个CommonJS/AMD模块格式解释器。这样浏览器就认识了define, exports,module这些东西，也就实现了模块化。

    2. browserify/webpack：是一种预编译模块打包的方案，不需要在浏览器中加载解释器。你在本地直接写JS，不管是AMD/CMD/ES6风格的模块化，它都能认识，并且编译成浏览器认识的JS。

三、模块化方案的对比

方案 代码风格 支持别名 速度 内存占用 整合别的模块 警告 结论

Browserify +1 -1 0 0 0 0 0

正则表达式学习笔记

发表于 2017-05-19 | 分类于 学习 | 阅读次数

一、什么是正则表达式

    正则表达式是由普通字符（括所有大写和小写字母、所有数字、所有标点符号和一些其他符号）以及特殊字符（称为”元字符”）组成的一种文字模式，它使用单个字符串来描述、匹配一系列匹配某个句法规则的字符串。

二、为什么使用正则表达式

    通过使用正则表达式，可以：
        1. 测试字符串内的模式。可以通过正则表达式进行数据验证。
        2. 替换文本。可以使用正则表达式来识别文档中的特定文本，完全删除该文本或者用其他文本替换它。
        3. 基于模式匹配从字符串中提取子字符串。可以查找文档内或输入域内特定的文本。

三、正则表达式的使用

1.普通字符

    普通字符包括没有显式指定为元字符的所有可打印和不可打印字符。这包括所有大写和小写字母、所有数字、所有标点符号和一些其他符号。

2.非打印字符

字符 描述
\cx 匹配由x指明的控制字符。
\f 匹配一个换页符。
\n 匹配一个换行符。
\r 匹配一个回车符。
\s 匹配任何空白字符，包括空格、制表符、换页符等等。
\S 匹配任何非空白字符。
\t 匹配一个制表符。
\v 匹配一个垂直制表符。

3.特殊字符

限定符

字符 描述
* 匹配前面的子表达式零次或多次。
+ 匹配前面的子表达式一次或多次。
? 匹配前面的子表达式零次或一次。
{n} n 是一个非负整数。匹配确定的 n 次。
{n,} n 是一个非负整数。至少匹配n 次。
{n,m} m 和 n 均为非负整数，其中n <= m。最少匹配 n 次且最多匹配 m 次。

定位符

字符 描述
^ 匹配输入字符串开始的位置。如果设置了 RegExp 对象的 Multiline 属性，^ 还会与 \n 或 \r 之后的位置匹配。
$ 匹配输入字符串结尾的位置。如果设置了 RegExp 对象的 Multiline 属性，$ 还会与 \n 或 \r 之前的位置匹配。
\b 匹配一个字边界，即字与空格间的位置。
\B 非字边界匹配。

如果100，刚好四分之一

发表于 2017-05-12 | 分类于 随笔 | 阅读次数

    时间向前望，总是没有尽头；向后看，却恍如昨日。不知不觉，已经工作两年了；不知不觉，已经二十五了；不知不觉，爸妈都已经老了；不知不觉……曾以为的漫漫人生路，不知不觉中也已经走了四分之一。回想一下，我这四分之一的人生，好像还挺有意思的。
    1992年12月11日凌晨5点左右，我来到了这个世界。然后一片空白的过了五年（五岁之前没什么印象，翻照片好像我爸天天带我去儿童公园），应该是一个无（没）忧（心）无（没）虑（肺）的幼年吧。第六年，我开始了学习生涯。幼儿园的两年，好像也只记得是在哪里上的，至于学了什么，毫无印象。
    终于终于，到了真正开始学习的年纪，我还真学到了让我毕生难忘的东西——乘法口诀表。从一年级抄到了二年级，所以别问我数学什么老师教的，正宗的数学老师啊。三年级开始，好像对数学没什么印象了（应该是太简单了），我只记得从那时候起，语文老师开启了背古诗、背课文还有背作文模式。现在想想真是苦逼，天天背着不知道什么东西的东西，我只想说，我经历过绝望。哦，对了，那年开始不只有语文和数学了。体育课，那时候我最讨厌的（真是犯贱），每节课都要跑步，然后我每次都是跑得最慢的，所以上什么体育课嘛。还有美术课，我记得我还进了美术组，什么国画、水彩画、书法样样精通（曾经我也是艺术家，哈哈哈）。到了五年，我第一次知道班主任原来可以是男的。从此，我们的语文课都是体育课（天天打乒乓）。也是那年，我知道了会跳舞的不一定是女的。跳了一个暑假之后，终于等到了小学打架无敌的一个年级——六年级。我是六年级的，谁敢跟我打架，哈哈哈，好蠢。
    到了初中，欢快的童年突然消失了，迎面而来的是无尽的作业。还记得小学升初中的那次考试成绩出来的那天晚上，我爸拿着我的卷子把我骂的狗血淋头。当时我就慌了，只能老老实实地，认认真真地埋头苦学。混着混着，到了初二，我学会了抄作业。那时候想着，卧槽，做作业还能这么轻松啊，然后，我就抄了整个初二的作业…说来也奇怪的，初中三年我成绩最好的就是初二。还记得那个时候抄作业不专业，经常被老师抓到。有一次被抓到，把我叫去办公室训了半天，然后无奈地跟我打了个赌——下次考试，我考进全校前十，就不管我抄作业了；考不进，就让我老老实实地做作业。结果那次考试成绩出来的时候，我惊呆了，刚好第十名。虽然之后抄作业还是要被抓，但这个逼，我能吹一辈子，哈哈哈。吹着吹着，初三到了，气氛瞬间就不一样了，每个人都开了挂地学习，搞得我也不敢抄作业了。但是对于只有抄作业buff的我来说，开启学霸模式好像并没什么软用，成绩越学越差，作业也是做得我怀疑人生。吹出去的萧中，最后也只能含泪奔向了三中。

    写着写着，还挺有意思的。有点晚了，睡觉先，明天继续！
